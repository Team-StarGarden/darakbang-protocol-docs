# 이름 규칙

## 공통

- 휴먼 미스를 피하기 위해 복수형(-es, -s)을 사용하지 않습니다. 그 대신에 List, Set 등을 뒤에 붙입니다.

## 패킷

- `PascalCase` 로 이름을 짓습니다.

## 클라이언트 바운드 패킷

- 패킷 이름 앞에 `I will` 이 붙어있다고 생각하고 이름을 짓습니다.

  e.g. `StartGame`, `Quit`, `Ready` 등

## 통보형 패킷

- `주어(생략가능) + 동사과거`의 형식으로 이름을 짓습니다.

  e.g. `PlayerJoined`, `Noticed` 등
